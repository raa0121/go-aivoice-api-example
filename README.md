# go-aivoice-api-example

[A.I. VOICE Editor API](https://aivoice.jp/manual/editor/api.html) を Go で実装するサンプル  
できるだけC++の実装をなぞりました。

## Usage

```
> go-aivoice-api-example
```

## Requirements
[A.I. VOICE](https://aivoice.jp) 1.3.0 以降

## Installation

go install github.com/raa0121/go-aivoce-api-example
